# Canonical Public Platform Records

Each immediate child directory is one governed record collection:

- audits
- findings
- claims
- sources
- source-captures
- calculations
- unknowns
- methodologies
- challenges
- responses
- corrections
- releases
- artifacts
- institutions

Each JSON file contains one record and is named with that record's permanent identifier. Generated website data does not belong here.

Public forms, browser code, and generated-site processes must not write to this directory.
