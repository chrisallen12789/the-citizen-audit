# Constitution Guardian

Status: v1 foundation

The Constitution Guardian is the governance inspection layer for The Citizen Audit.

It exists to keep agents, proposals, contributors, and future automation subordinate to institutional doctrine, operating principles, authority levels, and permission rules.

## Purpose

The Constitution Guardian is the first institutional component designed to say no.

It does not decide truth. It checks whether the institution's own rules are being respected before work proceeds.

## v1 scope

Constitution Guardian v1 checks:

- required doctrine files exist;
- authority levels exist;
- permission rules exist;
- active agents declare allowed and forbidden actions;
- Level 4 actions require human approval;
- proposals requiring high authority declare review status;
- core doctrine is represented in memory.

## Forbidden behavior

The guardian may not silently rewrite doctrine, weaken standards, edit conclusions, or approve truth-bearing changes.

## Future scope

Later versions should inspect diffs, pull requests, agent requests, and proposed record changes before execution.
