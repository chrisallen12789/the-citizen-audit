Manual Archive Workflow

Archive Assistant

This utility keeps the archive methodology strict while reducing repetitive operator work. It does not fabricate captures, hashes, or statuses, and it only exports what you explicitly confirm.

  • Load `archive_manifest.csv`.
  • Open the canonical URL and the Save Page Now URL for each record.
  • If Save Page Now fails, use Check Existing Archives to look for older Wayback snapshots.
  • Paste the returned archive URL or select a real snapshot. If Capture Date is blank, the assistant uses the Wayback timestamp first and only falls back to today when needed.
  • If the archive attempt fails and no real snapshot exists, document the failure state. That validates as a documented archive failure, but it does not create archive evidence.
  • Finish verifier names, review the auto-status, and export the updated manifest regularly.
  • Recommended setup: run npm run archive:assistant and use the localhost URL instead of opening this file directly from disk.
