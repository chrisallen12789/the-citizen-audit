Operating Standard

This site is designed to be read like a civic research institution, not a repository of files or opinion fragments.

Evidence first, not personality first.

The work is organized around claims, sources, decision rules, and open questions so readers can test the record directly.

Known-minimum, not guesswork.

When public reporting stops, the gap is named instead of padded with unsupported precision or hidden in rhetoric.

Reading Path

New visitors should be able to reach the right next step quickly.

1. Start Here

4. Source Library

5. Full Audit

6. Challenge The Audit
